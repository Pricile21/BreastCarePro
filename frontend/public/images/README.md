# Illustrations pour les articles

Ce dossier contient les illustrations utilisées dans les articles éducatifs.

## Images nécessaires

Pour l'article "Qu'est-ce que le cancer du sein?", les illustrations suivantes sont référencées :

1. `/images/anatomie-sein.png` - Anatomie du sein montrant les différents types de tissus
2. `/images/types-cancer-sein.png` - Schéma comparatif des différents types de cancer du sein
3. `/images/signes-symptomes.png` - Illustration des signes et symptômes à surveiller
4. `/images/methodes-depistage.png` - Méthodes de dépistage (autopalpation, mammographie, échographie)

## Format recommandé

- Format : PNG ou JPG
- Résolution : 800x600px minimum (ratio 4:3 recommandé)
- Poids : < 500KB par image pour optimiser le chargement mobile
- Style : Illustrations médicales claires, schémas explicatifs, infographies

## Note

Les images sont optionnelles. Si une image n'existe pas, un placeholder sera affiché automatiquement.
